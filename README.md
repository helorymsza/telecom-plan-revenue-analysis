# Telecom Plan Revenue Analysis

This project analyzes user behavior and revenue for the Megaline telecom company, comparing Surf and Ultimate plans and regional differences (NY-NJ vs Others).

## How to run locally
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Open the notebook: `jupyter notebook`
